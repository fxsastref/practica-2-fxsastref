# 1. Demostració d'elements de la sintaxi bàsica

```markdown
# Títol nivell 1

## Títol nivell 2

### Títol nivell 3

Aquest és un text amb **negreta**, *cursiva* i ~~ratllat~~.

### Llistes

Llista amb punts:

- Element 1
- Element 2
- Element 3

Llista numerada:
1. Primer element
    1. Primer subelement
    2. Segon subelement
2. Segon element
3. Tercer element
```

i aquest és el resultat:

> # Títol nivell 1
> ## Títol nivell 2
> ### Títol nivell 3
> Aquest és un text amb **negreta**, *cursiva* i ~~ratllat~~.
> ### Llistes
> Llista amb punts:
>   - Element 1
>   - Element 2
>   - Element 3
> 
>  Llista numerada: 
>   1. Primer element 
>     1. Primer subelement
>     2. Segon subelement
>   2. Segon element
>   3. Tercer element

---

- Els títols s'escriuen amb el símbol `#`, un nombre variable segons el nivell (més símbols, nivell més baix).
- La negreta es fa posant el text entre dos asteriscs dobles `**`.
- La cursiva es fa posant el text entre un asterisc simple `*`.
- El ratllat es fa posant el text entre doble tilde `~~`.
- Les llistes amb punts s’inicien amb `-` o `*`.
- Les llistes numerades s’inicien amb un nombre seguit de punt i espai.
