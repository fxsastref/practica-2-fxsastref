# 2. Enllaços i imatges

## Enllaços externs

Per crear un enllaç extern, s'escriu el text entre corxets `[ ]` i la URL entre parèntesis `( )`:

[Visita Google](https://www.google.com)

També pots afegir un títol que apareix quan passes el cursor, amb cometes després de la URL:

[Visita GitHub](https://github.com "Plataforma de desenvolupament")

## Imatges externes

Per a inserir una imatge externa, s'utilitza una sintaxi semblant a l'enllaç però amb una exclamació `!` al principi:

![Logo de GitHub](https://github.githubassets.com/images/modules/logos_page/GitHub-Mark.png)

## Imatges locals

Si tens una imatge dins un directori local anomenat `img`, la pots referenciar així:

![Imatge local exemple](img/Markdown-mark.svg "LogoMarkdown")
